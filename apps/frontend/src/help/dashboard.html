Dashboard guide

Use the dashboard as your daily control center.

This page brings your most important modules into one overview so you can decide what needs attention before diving into a dedicated page.

What this page is for

The dashboard is a summary page. It does not replace the feature pages, but it helps you scan tasks, habits, focus, finance, calories, and admin information in one place.

Important areas

  • Tasks widget: quick view of current commitments and scheduling pressure.
  • Habits widget: a compact snapshot of routines and streak momentum.
  • Focus widget: a high-level reminder of attention and work rhythm.
  • Finance widget: a quick read on money flow and current period performance.
  • Calories widget: fast nutrition and intake visibility.
  • Admin widget: only appears for admin users.

How to use it well

  1. Start here to understand what needs action today.
  2. Open the detailed page from the left navigation when a widget shows something worth fixing.
  3. Use this page as a morning and evening checkpoint rather than a place for detailed editing.

Good to know

The Focus route currently points to the dashboard, so its help content is the same on purpose.
