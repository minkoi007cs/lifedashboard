Finance guide

Track income, expenses, and pay-period profit.

Finance Manager is designed for short-cycle money tracking, with fast daily entry and a persistent side summary.

What this page is for

Use this page to record daily financial activity and keep a simple view of how the current pay period is performing.

Primary tabs

  • Dashboard: shows the main financial summaries and trends.
  • Daily Entry: captures new numbers for the day.

Important layout behavior

The right column keeps the pay-period summary visible while the main content changes. That makes it easy to enter numbers without losing context.

Recommended workflow

  1. Open Daily Entry and log the day's values.
  2. Return to Dashboard to review impact.
  3. Use the side summary as the quick truth for the current pay period.
